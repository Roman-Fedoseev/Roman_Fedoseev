Решение для задачи предсказания ценовых интервалов (5-й и 95-й квантили) с оптимизацией метрики IoU (Intersection over Union) между истинным и предсказанным интервалами.

Ключевая идея решения — раздельное моделирование нижней и верхней границы, а затем калибровка ширины интервала на валидационной выборке.

Основные идеи решения
Отдельные модели для p05 и p95
Продвинутый feature engineering с агрегациями по train
Подбор гиперпараметров (Optuna) отдельно для каждой модели
Калибровка интервала через перебор квантилей nonconformity scores
Оптимизация напрямую под IoU
